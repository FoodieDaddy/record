# 项目概述

## 产品概述

太空记分器（Space Scorekeeper，工程代号 smartrecord）是一个多人编队记分与航迹复盘微信小程序。产品世界观为「太空方舟」：每个用户是一艘个人航船；编队是一组个人航船为了完成同一段短程任务而组成的临时航行单元；任务过程中产生的读数会写入轨迹，封存后进入航迹档案，继续驱动今日指令、镜像投影和身份沉淀。

产品主线：任务编队 → 脉冲记录 → 航迹复盘 → 指令推演 → 镜像投影 → 身份沉淀。

一句话世界观：

```text
每个用户是一艘个人航船。
编队不是普通房间，而是一组个人航船为了完成同一段短程任务而组成的临时航行单元。
脉冲不是普通积分，而是任务航程中的核心读数单位。
它记录任务推进、链路流向、阶段结果和成员之间的协作变化。
每一次脉冲变化都会成为航迹点。
当主控封存航程后，所有航迹点写入航迹档案，继续驱动导航舱、全息舱和识别舱。
```

用户可见短版：

```text
编队不是普通房间，而是一段任务航程。
脉冲不是普通积分，而是航程中的任务读数。
每一次脉冲变化都会写入轨迹，封存后成为航迹样本。
```

## 产品定位

太空记分器不是普通计分表，也不是传统表单工具。首先必须让用户一眼知道这是记分器，其次通过太空编队、驾驶舱、航迹档案形成差异化。整体气质：黑底、蓝光、克制、冷静、数据感、舰载终端感、复盘感、身份感。

禁止做成：微信默认表单页、传统设置页、娱乐押注工具、玄学工具、儿童化游戏界面、大红大金视觉、满屏营销卡片、浮夸科幻游戏界面。

技术栈、启动方式、端口规划详见 CODEBUDDY.md。

## 四个主 Tab

| Tab | 舱位定位 | 主要体验 |
|---|---|---|
| 编队 | 驾驶舱 | 创建任务编队、展开信标、加入编队、写入脉冲、封存航程 |
| 指令 | 导航舱 | 读取航迹样本、生成今日指令、查看推进节奏与安全边界 |
| 镜像 | 全息舱 | 校准人格协议、查看镜像投影、协议一致率与全息扫描 |
| 身份 | 识别舱 | 管理本舰呼号、识别徽标、授权等级、通讯/视觉/触感协议 |

## 最终体验目标

```text
我在驾驶舱创建编队，发起一段任务航程。
我展开信标邀请成员加入。
我通过不同脉冲记录协议写入脉冲——流向、推进、同步或航段读数。
我封存航程，所有脉冲轨迹写入航迹档案。
我在导航舱生成今日指令。
我在全息舱查看镜像。
我在识别舱管理本舰身份。
每一次封存，都会写入航迹档案，让系统更懂我。
```

所有页面必须服务于：任务航程 → 脉冲记录 → 航迹复盘 → 指令推演 → 镜像投影 → 身份沉淀。
