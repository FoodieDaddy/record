# 验收清单

## 全局验收

- [ ] 所有主操作为中文，英文仅弱装饰。
- [ ] 运行时不出现审核高风险词。
- [ ] 运行时不出现技术穿帮词：`LLM`、`LOW-NOISE`、`HIGH_RISK`、`fallback` 等。
- [ ] 红色只用于危险、错误、失败。
- [ ] 所有核心页面支持 reduce-motion。
- [ ] 所有 timer / interval / animation frame 在页面隐藏或卸载时清理。
- [ ] 不新增高频轮询、无差别广播、运行期大对象反复序列化。
- [ ] 底部导航显示为：编队 / 指令 / 镜像 / 身份。
- [ ] 选中态通过图标点亮和能量条表达，不靠多行文字。
- [ ] 编队邀请、二维码、分享入口和扫码加入统一称为「信标」，不能全部替换为「编队码」。
- [ ] 「编队码」只表示 6 位短码。
- [ ] 运行时不出现「黑匣子」，统一为「航迹档案 / 航迹样本」。
- [ ] 运行时不出现「五维扫描」，统一为「全息扫描」。
- [ ] 运行时不出现「今日策略 / 生成策略」，统一为「今日指令 / 导航计算」。
- [ ] 运行时不出现「航行核心」，统一为「导航核心」。
- [ ] 运行时不出现「起飞甲板」，统一为「导航舱」。
- [ ] 运行时不出现「舰员代号」，统一为「本舰呼号」。
- [ ] 运行时不出现「头盔识别」，统一为「识别徽标」。
- [ ] 运行时不出现「档案舱」，统一为「识别舱」。
- [ ] 运行时不出现「数据矩阵」，识别舱已收敛为「本舰状态」。
- [ ] 运行时不出现「身份档案」，统一为「识别档案」。
- [ ] 运行时不出现「船坞 / 船籍」。
- [ ] 用户称呼体系：当前用户称「本舰」，任务参与者称「编队成员」。

## 编队页验收

```text
我在驾驶舱创建一个任务编队。
我能展开信标邀请成员加入。
我能加入编队，看到编队席位和脉冲轨迹。
我能封存航程，并理解样本写入航迹档案。
```

## 指令页验收

```text
我进入导航舱启动导航核心。
我看到今日指令投影，而不是普通 AI 建议。
我先获得舰载指令，再查看状态读数、推进节奏和安全边界。
重新计算会回到待机页，由我再次手动计算。
```

## 镜像页验收

```text
我进入全息舱查看镜像投影。
系统读取航迹样本，展示人格协议、协议偏移和全息扫描。
我能查看航迹档案摘要（已写入样本、最近航程、封存时间）。
我能从全息舱进入航迹回放。
判读短、冷静、像系统扫描结果。
```

## 身份页验收

```text
我在识别舱管理本舰档案。
本舰呼号是页面视觉核心。
识别舱不出现数据矩阵和航迹档案摘要。
本舰状态只展示航行经验和稳定读数。
声音、动效、触感像装备协议，而不是普通设置开关。
```
