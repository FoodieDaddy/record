# 验收清单

## 全局验收

- [ ] 用户可见品牌统一为「太空记分器 / Space Scorekeeper」，不再使用 Smart Record 或脉冲终端作为主品牌。
- [ ] 用户第一眼能理解这是多人记分器，不是普通科幻终端。
- [ ] 底部导航：编队 / 指令 / 镜像 / 身份；选中态通过图标点亮 + 能量条表达。
- [ ] 所有核心页面支持 reduce-motion；timer/interval/rAF 在页面隐藏或卸载时清理。
- [ ] 不新增高频轮询、无差别广播、运行期大对象反复序列化。
- [ ] 红色只用于危险、错误、失败、解散、撤离。
- [ ] 运行时不出现在审核高风险词、技术穿帮词（LLM / LOW-NOISE / HIGH_RISK / fallback 等）、产品语言避用词（战斗/攻击/打击/伤害/击败/对手/敌人/胜负/输赢/决斗/PK）。

### 术语统一

- [ ] 编队邀请/分享入口/扫码加入 =「信标」；「编队码」仅指 6 位短码。
- [ ] 「航迹档案 / 航迹样本」替代「黑匣子」。
- [ ] 「全息扫描 / 全息图谱」替代「五维扫描 / 五维图谱 / 雷达图」。
- [ ] 「今日指令 / 导航计算 / 指令卡」替代「今日策略 / 生成策略 / 策略卡」。
- [ ] 「协议校准」替代「人格测试」；「镜像投影」替代「行为画像」。
- [ ] 「识别徽标」替代「头盔识别」；「本舰呼号」替代「舰员代号」。
- [ ] 「断开终端」替代「退出登录」；「识别舱」替代「档案舱」。
- [ ] 「编队 / 编队码」替代「房间 / 房间号」。
- [ ] 不出现：预测/预知/运势/神谕/占卜、船坞/船籍、航行核心、舰载指令、重新点火、成员协议、全息观测舱、起飞甲板、身份档案、数据矩阵。

## 编队页验收

- 驾驶舱一体成型（`cockpit-shell-v2` 上方舷窗 + 下方驾驶台），非多卡片堆叠。
- HUD 仪表显示：本舰脉冲、编队码、成员数、记录模式、航程阶段、链路状态。
- 首屏下方主操作按钮完整可见，不被底部 Dock 遮挡。
- AR 悬浮按钮（信标 / 航迹）：半透明全息面板风格，纯 CSS 图标，76-92rpx 高。
- 脉冲流向面板（`pulse-vr-cluster`）：AR 浮空风格，宽度不超出屏幕，3/6/9/确认按钮不被裁剪，底部间距 `calc(170rpx + env(safe-area-inset-bottom))`。
- 封存航程为主流程按钮（非大红按钮），解散编队为二级危险操作。
- 航迹档案在驾驶舱为轻量入口，详情进全息舱。
- 脉冲是任务读数，运行时不出现在对抗词。

## 指令页验收

- 进入导航舱开始导航计算；看到今日指令投影。
- 重新计算回到导航待机，需手动再计算。
- 不出现预测/玄学/运势/神谕/策略旧词。
- 生成中导航核心保持可见，不长时空黑屏。

## 镜像页验收

- 页面标题「全息舱」；首屏：镜像投影 + 协议一致率 + 航迹样本 + 全息扫描图。
- 协议分析默认折叠，可展开。
- 重新校准 / 修改协议 / 生成镜像图片按钮完整显示，不被底部导航遮挡。
- 20 题校准流程按钮完整显示，不被 TabBar / Home Indicator 遮挡；完成 Toast「校准已完成」。
- 镜像图片为独立排版，非页面截图。
- 可查看航迹档案摘要，从全息舱进入航迹回放。
- 不出现全息观测舱/黑匣子/五维扫描/雷达图/人格测试/预测/预知/运势。

## 身份页验收

- 页面标题「识别舱」；本舰呼号为视觉核心，识别徽标可点击修改。
- 装备协议完整显示，不被底部导航遮挡。
- 「断开终端」按钮完整显示，不被 TabBar / Home Indicator 遮挡。
- 不出现舰员档案/舰员代号/头盔识别/数据矩阵/退出登录/个人中心/我的。
