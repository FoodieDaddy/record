# 验收清单

## 全局验收

- [ ] 所有主操作为中文，英文仅弱装饰。
- [ ] 运行时不出现审核高风险词。
- [ ] 运行时不出现技术穿帮词：`LLM`、`LOW-NOISE`、`HIGH_RISK`、`fallback` 等。
- [ ] 红色只用于危险、错误、失败。
- [ ] 所有核心页面支持 reduce-motion。
- [ ] 所有 timer / interval / animation frame 在页面隐藏或卸载时清理。
- [ ] 不新增高频轮询、无差别广播、运行期大对象反复序列化。
- [ ] 底部导航显示为：编队 / 指令 / 镜像 / 身份。
- [ ] 选中态通过图标点亮和能量条表达，不靠多行文字。
- [ ] 编队邀请、二维码、分享入口和扫码加入统一称为「信标」，不能全部替换为「编队码」。
- [ ] 「编队码」只表示 6 位短码。
- [ ] 运行时不出现「黑匣子」，统一为「航迹档案 / 航迹样本」。
- [ ] 运行时不出现「五维扫描」，统一为「全息扫描」。
- [ ] 运行时不出现「今日策略 / 生成策略 / 策略卡」，统一为「今日指令 / 导航计算 / 指令卡」。
- [ ] 运行时不出现「航行核心」，统一为「导航舱 / 导航计算」。
- [ ] 运行时不出现「舰载指令」，统一为「今日指令」。
- [ ] 运行时不出现「重新点火 / 点火航行核心」，统一为「重新计算 / 开始导航计算」。
- [ ] 运行时不出现「成员协议」，统一为「航迹协议」。
- [ ] 运行时不出现「预测 / 预知 / 运势 / 神谕 / 占卜」。
- [ ] 运行时不出现「全息观测舱」，统一为「全息舱」。
- [ ] 运行时不出现「五维图谱」，统一为「全息图谱」。
- [ ] 运行时不出现「人格测试」，统一为「协议校准」。
- [ ] 运行时不出现「行为画像」，统一为「镜像投影」。
- [ ] 运行时不出现「起飞甲板」，统一为「导航舱」。
- [ ] 运行时不出现「舰员代号」，统一为「本舰呼号」。
- [ ] 运行时不出现「头盔识别」，统一为「识别徽标」。
- [ ] 运行时不出现「档案舱」，统一为「识别舱」。
- [ ] 运行时不出现「数据矩阵」，识别舱已收敛为「本舰状态」。
- [ ] 运行时不出现「身份档案」，统一为「识别档案」。
- [ ] 运行时不出现「船坞 / 船籍」。
- [ ] 用户称呼体系：当前用户称「本舰」，任务参与者称「编队成员」。

## 编队页验收

```text
我在驾驶舱创建或加入一个任务编队。
首屏是一屏驾驶舱：`cockpit-shell-v2` 里的上方舷窗 + 下方驾驶台一体成型，不再是多张卡片堆叠。
我能透过舷窗看到外部编队航船或等待编队接入状态。
我能通过 HUD 仪表读取本舰脉冲、编队码、编队成员数、记录模式、航程阶段和链路状态。
我不会在驾驶舱主视觉里看到自己的头像大卡片。
我能展开信标邀请成员加入。
我能点击外部航船记录脉冲。
我能在首屏下方完整看到「记录脉冲 / 展开信标」按钮，不被底部 Dock 遮挡。
我能封存航程，并理解样本写入航迹档案。
航迹档案在驾驶舱只作为轻量入口，详细航迹映射进入全息舱查看。
```

## 指令页验收

```text
我进入导航舱开始导航计算。
我看到今日指令投影，而不是普通 AI 建议。
我先获得今日指令，再查看状态读数、推进节奏和安全边界。
重新计算会回到导航待机，由我再次手动计算。
我不会看到预测、玄学、运势、神谕或策略旧词。
生成中导航核心保持可见，不会出现长时间空黑屏。
```

## 镜像页验收

```text
我进入全息舱查看镜像投影，页面标题为「全息舱」。
首屏展示镜像投影、协议一致率、航迹样本和全息扫描图，不堆满分析卡片。
协议分析（系统判读、信号标签、协议偏移、协议演化）默认折叠，可展开查看。
我能点击「重新校准」按钮，按钮完整显示不被底部导航遮挡。
我能点击「修改协议」按钮，按钮完整显示不被底部导航遮挡。
我能点击「生成镜像图片」按钮，按钮完整显示不被底部导航遮挡。
镜像图片可以生成预览，并提供保存、发送给朋友和关闭预览。
镜像图片是独立排版，不是页面截图，文案不出现旧词。
20 题校准流程中的下一步/退出/完成按钮完整显示，不被底部导航遮挡。
校准提交面板不被底部导航遮挡。
校准流程底部按钮不被 Home Indicator 遮挡。
校准退出确认弹窗按钮完整显示。
校准完成 Toast 为「校准已完成」。
校准失败提示为「校准失败，请稍后重试」。
修改协议流程中的确认按钮完整显示，不被底部导航遮挡。
页面底部滚动到最后时，最后一个模块不被 TabBar 或 Home Indicator 遮挡。
我能查看航迹档案摘要（已写入样本、最近航程、封存时间）。
我能从全息舱进入航迹回放。
判读短、冷静、像系统扫描结果。
运行时不出现「全息观测舱」「黑匣子」「五维扫描」「雷达图」「人格测试」等旧词。
运行时不出现预测、预知、运势、神谕等表达。
```

## 身份页验收

```text
我在识别舱查看本舰档案，页面标题为「识别舱」。
本舰呼号是页面视觉核心，识别徽标可点击修改。
授权等级和航行经验清晰展示。
装备协议（通讯/视觉/触感/音色）完整显示，不被底部导航遮挡。
「断开终端」按钮完整显示，不被底部导航或 Home Indicator 遮挡。
退出确认弹窗按钮完整显示。
识别舱不出现数据矩阵和航迹档案摘要。
授权状态只展示航行经验和稳定读数。
声音、动效、触感像装备协议，而不是普通设置开关。
退出后登录页接入按钮不被 Home Indicator 遮挡。
运行时不出现「舰员档案/舰员代号/头盔识别/数据矩阵/退出登录/个人中心/我的」。
```
