# 内容安全

运行时页面、Prompt、备用指令、分享图文、Canvas 海报、Toast、弹窗、按钮、空状态、错误态必须避开审核高风险词。文档可列出禁词用于约束，但不得把禁词复制到用户可见页面。

## 产品语言避用词

运行时用户可见内容禁止出现：

```text
战斗、攻击、打击、伤害、击败、对手、敌人、胜负、输赢、决斗、PK
```

替换为：任务、航程、读数、脉冲、流向、写入、推进、同步、航段、门限、回稳、封存、航迹。

## 品牌规则

- 用户可见品牌：「太空记分器」或「Space Scorekeeper」。
- 禁止在用户可见主品牌使用「Smart Record」或「脉冲终端」。
- 禁止用收益/预测/输赢/战斗类表达解释记分。

## 技术穿帮词禁止直出

可存在于代码变量/接口字段/日志/内部注释，不得出现在运行时用户可见内容中：

```text
LLM、LOW-NOISE、MEDIUM-NOISE、HIGH-NOISE、HIGH_RISK、fallback、oracle、fortune、strategy output、THE CALIBRATOR、校准者
```

展示映射：LLM→主引擎、LOW-NOISE→低噪、MEDIUM-NOISE→中噪、HIGH-NOISE→高噪、HIGH_RISK→偏高、fallback→备用指令、strategy output→指令投影、THE CALIBRATOR/校准者→今日指令。

## 审核高风险词

运行时用户可见内容禁止出现：

```text
棋牌、赌博、赌、下注、押注、筹码、牌局、牌桌、打牌、麻将、扑克、德州、梭哈、赢钱、赚钱、发财、稳赚、必胜、翻本、追损、运势、算命、占卜、塔罗、抽牌、神谕、卦象、黄历、风水、开运、转运、改运、预测输赢、胜率提升
```

策略/镜像/分享卡等额外禁止：

```text
盈利、亏损、收益、胜率、预测、稳赚、暴富、回本、翻盘、赔率、盘口、庄家
```

替换为：任务/航段/场景、脉冲/任务读数、结果波动、状态读数、今日指令投影、导航计算、指令参考、状态分析、节奏校准、情绪化修正、安全边界、航迹样本。

## 输出过滤流程

1. 主引擎输出过敏感词 → 命中则丢弃，使用静态备用指令。
2. 前端二次 sanitize（`title`/`subtitle`/`tags`/`verdict`/海报/分享标题）。
3. 分享卡/Canvas 海报/Toast/弹窗/空状态/错误态均需过同一套敏感词 + 世界观映射。
4. 推荐语气：冷静、克制、短句、舰载终端感、复盘视角。

## 指令与镜像内容边界

- 指令只能给状态管理/节奏提醒/安全边界/行为复盘视角；禁止预测/玄学/收益承诺/AI 长文；不出现预知/预测/运势/占卜/神谕。
- 镜像判读短、冷静、数据感强；不写大段心理鸡汤，不做社交测试娱乐化。

## 页面术语映射

| 页面 | 禁止出现 | 统一为 |
|---|---|---|
| 全息舱 | 全息观测舱、黑匣子/黑匣子样本、五维扫描/五维投影/五维图谱/雷达图、人格测试、行为画像 | 全息舱、航迹档案/航迹样本、全息扫描/全息投影/全息图谱/全息扫描图、协议校准、镜像投影 |
| 识别舱 | 舰员档案/舰员代号/头盔识别、数据矩阵、退出登录/个人中心/我的、船坞/船籍舱/档案舱 | 本舰档案/本舰呼号/识别徽标、授权状态、断开终端 |
| 登录页 | 登录/微信登录/用户登录 | 接入/终端接入/接入终端 |
| 编队页 | 战斗/攻击/打击/伤害/击败/对手/敌人/胜负/输赢/决斗/PK、房间/房间号 | 任务/航程/读数/脉冲/流向/写入、编队/编队码 |

镜像 Canvas 海报可独立使用「SPACE SCOREKEEPER」品牌标识，不附加英文舱位副标题。

## 运行时错误展示

- 服务端错误由业务层统一转 `Result` + 可识别 code；前端只展示简短状态提示。
- 不展示堆栈、类名、连接串、JWT、微信 code、OSS 签名 query 或完整 WebSocket URL。
